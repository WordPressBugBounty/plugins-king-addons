# 360° Product Viewer

Render a spinable 360° sequence for products using ordered image frames with swipe or drag rotation.

## Free features
- Up to 36 frames with drag/swipe rotation and start-frame selection.
- Autoplay with adjustable speed, invert drag, and aspect-ratio control.
- Loader, drag hint, and frame progress overlays with basic styling controls.

## Pro features
- Unlimited frames with ping-pong autoplay, scroll-to-rotate, inertia, and multi-axis drag.
- Zoom toggle with adjustable scale plus expanded overlay styling.
- Frame-based hotspots with labels and optional links, including hotspot styling controls.

## How to use
1. Add the **360° Product Viewer** widget and upload sequential frame images in order.
2. Set aspect ratio, start frame, drag sensitivity, and optionally enable autoplay.
3. Tweak loader, hint, and progress text/styles. (Pro) Enable zoom, inertia/scroll rotation, and add hotspots aligned to specific frames.
