# Ajax Add to Cart

Ajax-powered WooCommerce add-to-cart button with success notice and styling controls.

## Free features
- Use current product (single) or specify product ID fallback.
- Fixed quantity, button text, success message.
- Alignment, typography, colors, border, radius; inline spinner and notice.

## Pro features
- Icons/badges on button, mini-cart fragment toggle.
- Custom success/redirect behaviors, quantity selector, variable products.

## How to use
1. Place the **Ajax Add to Cart** widget.
2. Enable “Use Current Product” on single product pages, or set a Product ID fallback.
3. Set quantity, button label, and success message.
4. Style the button and notice; test add-to-cart on frontend.
5. (Pro) Add icons/badges, advanced behaviors, and variable product handling.
