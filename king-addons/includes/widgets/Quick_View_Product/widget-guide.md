# Quick View Product (WooCommerce)

Opens a product in a modal for fast preview with image, price, rating, excerpt, and add-to-cart.

## Free features
- Use current product (single) or specify Product ID.
- Quick View trigger button, modal with loader, ESC/overlay/close handling.
- Styling for button, overlay, modal radius/shadow/background.

## Pro features
- Gallery thumbnails/zoom, variation selector inside modal.
- Sticky CTA, related products, custom trigger/badge options.

## How to use
1. Add the **Quick View Product** widget.
2. Enable “Use Current Product” on product pages or set a Product ID.
3. Set button text; style button, overlay, and modal container.
4. (Pro) Enable gallery/zoom, variations, and sticky CTA.
