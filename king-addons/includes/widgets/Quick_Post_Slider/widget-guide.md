# Simple Post Slider - Widget Guide

## Overview
Simple Post Slider displays WordPress posts in a responsive slider with safe default spacing so arrows and dots do not overlap content.

## Free Features
- Query controls: posts per page, order/orderby, category slugs, ignore sticky.
- Show/hide featured image, meta (date & author), excerpt with length, and read more button.
- Slider controls: slides per view (desktop/tablet/mobile), gap, loop, autoplay, speed.
- Navigation and pagination with inside/outside placement, alignment, offsets, and hide on tablet/mobile.
- Card/text/button styling, shadows toggles, alignments.

## Pro Features
- Arrow and pagination style presets.
- Advanced animations and extended placement controls (per Pro feature set).
- Additional badge shapes/offsets (if enabled in Pro pack).

## How to Use
1. Add “Simple Post Slider” to the page.
2. Set posts per page, order, and optional category slugs.
3. Toggle featured image, meta, excerpt, and read more; adjust excerpt length.
4. Configure slides per view per breakpoint, gap, autoplay, and speed.
5. Set navigation/pagination placement and hide on tablet/mobile if needed.
6. Style card, text, meta, and button; disable shadows if desired.
