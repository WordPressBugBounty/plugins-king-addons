# Breadcrumbs

Lightweight breadcrumb navigation with customizable separator, alignment, and typography.

## Free features
- Home toggle and custom label.
- Separator text, gap, alignment.
- Link/current colors, typography, optional current item.

## Pro features
- Icon separators, custom home icon/prefix.
- JSON-LD schema toggle.
- Max depth and advanced layouts.

## How to use
1. Drop the **Breadcrumbs** widget.
2. Configure Home label and separator.
3. Style links, current item, and alignment.
4. (Pro) Enable icons, schema, and depth controls as needed.
