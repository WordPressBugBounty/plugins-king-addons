# WooCommerce Floating Cart Icon - Widget Guide

## Overview
Floating WooCommerce cart button with badge and quick mini-cart preview that opens without leaving the page.

## Free Features
- Floating cart icon with left/right placement and offset controls.
- Optional label and count badge synced with WooCommerce cart.
- Quick summary shows first items, remaining count notice, and subtotal toggle.
- Direct View Cart and Checkout links inside the preview.

## Pro Features
- Hover or click trigger plus optional auto-open after add to cart.
- Drawer layout for wider previews with thumbnails.
- Adjustable item limit with remove links and thumbnails.
- Hide when cart is empty and custom labels for action buttons.

## How to Use
1. Add “WooCommerce Floating Cart Icon” to the page (WooCommerce required).
2. Choose side and offsets, pick an icon, and toggle label/badge visibility.
3. Enable subtotal display and style the button, badge, and panel.
4. (Pro) Switch trigger to hover, choose drawer layout, set item limit, and customize action labels.
5. Save and verify the preview opens and links lead to the cart or checkout pages.
