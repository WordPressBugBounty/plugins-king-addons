# Custom Post Types Grid

Display posts from selected post types in a responsive grid with optional featured image, meta, excerpt, and read-more link. Pro unlocks advanced queries and custom field output for any meta key.

## Free Features
- Choose post type (`post` or `page`)
- Limit up to 6 items with order and order direction
- Toggle featured image, meta, excerpt, read-more
- Set columns and gap, plus basic styling controls

## Pro Features
- Select any public custom post types (multi-select)
- Higher limits, offset, include/exclude IDs
- Taxonomy filters via `taxonomy:slug-one,slug-two`
- Order by custom meta (text or number) with `meta_key`
- Custom Fields repeater with label, prefix/suffix, and fallback

## Usage Tips
- For taxonomy filters, separate each taxonomy group with `;` and slugs with commas (e.g., `category:news,updates; portfolio_cat:web`).
- When ordering by custom fields, set the `Custom Field Key` and choose the appropriate order type.
- Card click is disabled in the Elementor editor to avoid interfering with editing.
