# Compare Products Table (WooCommerce)

Side-by-side product comparison table with images, pricing, rating, stock, and add-to-cart.

## Free features
- Manual product selection via IDs (up to 4).
- Rows: image, price, rating, SKU, stock, short description, add-to-cart.
- Layout: cell alignment, padding.
- Styling: header and body typography/colors/borders/shadow, button styling.

## Pro features
- Unlimited products and category filters.
- Hide identical rows, sticky header.
- Extra rows: dimensions, weight, categories, tags.
- Difference highlighting and advanced styling.

## How to use
1. Add **Compare Products Table**.
2. Enter product IDs (comma-separated); set max products (Free up to 4).
3. Toggle which rows to show (price/rating/SKU/stock/description/button).
4. Adjust alignment, padding, and style header/body/button.
5. (Pro) Enable category filter, hide identical rows, sticky header, and extra attributes.
