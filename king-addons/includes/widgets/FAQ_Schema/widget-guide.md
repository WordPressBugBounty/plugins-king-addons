# FAQ Schema

Accordion FAQ with valid schema.org FAQPage JSON-LD.

## Free features
- Repeater items: question + answer (WYSIWYG).
- Optional open-first item.
- Styling: question/answer typography and colors, item background/border/radius/shadow, spacing/padding.
- Generates valid FAQPage schema with Question/Answer (sanitized).

## Pro features
- Single-open mode, expand/collapse all button.
- Search filter, custom open/close icons, simple animations.
- Option to disable schema output (for special cases).

## How to use
1. Add **FAQ Schema**, create Q/A items.
2. Toggle open-first, adjust spacing and styles.
3. Check schema output (FAQPage) in page source.
4. (Pro) Enable single-open, search, toggle-all, custom icons, animations, or disable schema if needed.
