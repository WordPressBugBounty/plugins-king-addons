# Event Calendar

Responsive event list with date badge, meta, and CTA. Mobile-friendly stacked layout.

## Free features
- Repeater events: title, start/end date-time, time label, location, category, color, description, link.
- Toggles: show time, location, description, CTA button (with custom text).
- Styling: card background/border/radius/shadow/padding; badge styles; typography and colors for title/meta/description/button.
- Sorted by start date ascending.

## Pro features
- Filters: category dropdown, search, upcoming-only, date range.
- Compact layout option and hover animations.
- Category chip coloring and ongoing-highlight.

## How to use
1. Add **Event Calendar** and create events in the repeater (set start/end date-time).
2. Fill time label/location/category/color/description/link as needed.
3. Toggle visibility of time/location/description/button.
4. Style card, badge, title/meta/description/button.
5. (Pro) Enable filters, upcoming-only, date range, and compact/animation options.
