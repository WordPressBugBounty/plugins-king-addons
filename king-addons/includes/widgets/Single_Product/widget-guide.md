# Single Product (WooCommerce)

Displays a single WooCommerce product with image, price, rating, excerpt, and add-to-cart.

## Free features
- Use current product on single pages or pick Product ID fallback.
- Toggles for image, price, short description, rating, add-to-cart button.
- Layout gap/alignment, card border/radius/shadow; typography and colors for title/price/excerpt; button styling.

## Pro features
- Sale/badge overlays, stock bar, countdown, icons/quantity selector.
- Gallery thumbs/layouts and enhanced CTA behaviors.

## How to use
1. Add the **Single Product** widget.
2. Enable “Use Current Product” on product pages, or set Product ID.
3. Choose which parts to show (image, price, rating, excerpt, button).
4. Style card, text, and button.
5. (Pro) Enable badges/stock/countdown and gallery/quantity enhancements.
