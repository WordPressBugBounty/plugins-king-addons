# Custom Post Types Slider

Display posts from selected post types in a Swiper slider with featured images, meta, excerpts, and read-more links. Pro unlocks advanced queries and custom field output within slides.

## Free Features
- Choose post type (`post` or `page`)
- Limit up to 6 items; order and order direction
- Toggle featured image, meta, excerpt, read-more
- Slider basics: slides per view, space, speed, autoplay, loop, arrows, pagination
- Styling for card, text, meta, and button

## Pro Features
- Select any public custom post types (multi-select)
- Higher limits, offset, include/exclude IDs
- Taxonomy filters via `taxonomy:slug-one,slug-two`
- Order by custom meta (text or number) with `meta_key`
- Custom Fields repeater with label, prefix/suffix, fallback

## Usage Tips
- For taxonomy filters, separate each taxonomy group with `;` and slugs with commas (e.g., `category:news,updates; portfolio_cat:web`).
- When ordering by custom fields, set the `Custom Field Key` and choose the appropriate order type.
- Slide click is disabled in the Elementor editor to keep editing selection easy.
