# Sticky Video

Sticky Video keeps a compact video visible in a screen corner with a dismissible close button. Use it to keep YouTube, Vimeo, or self-hosted clips playing while visitors scroll.

## Free features
- YouTube/Vimeo ID or self-hosted video with start time, autoplay, mute, loop, and controls toggles.
- Fixed bottom-right/left placement with adjustable width, aspect ratio, and offsets.
- Optional label beneath the player plus close button toggle.
- Lightweight player wrapper with custom background, border, radius, and shadow styling.

## Pro features
- Top-corner placements with independent X/Y offsets.
- Delayed or scroll-triggered reveal plus device targeting.
- Persistent close state with configurable duration.
- CTA button under the player with link styling controls.

## How to use
1. Add the **Sticky Video** widget.
2. Pick a source (YouTube ID, Vimeo ID, or self-hosted file) and set start, autoplay, mute, loop, and controls.
3. Adjust width, aspect ratio, offsets, and choose a bottom corner. Style the container, close button, and optional label.
4. (Pro) Select top-corner placements, add scroll/delay triggers, device filters, and enable close persistence.
5. (Pro) Add a CTA label and link to drive clicks beneath the mini-player.
