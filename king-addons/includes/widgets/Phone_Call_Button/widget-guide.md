# Phone Call Button

Call-to-action button with phone icon and tel: link.

## Free features
- Phone number to tel: link, button text, icon select.
- Alignment, icon position.
- Styling: text/background colors, border/radius/shadow, padding, typography.

## Pro features
- Floating button (left/right), mobile-only or desktop-only visibility.
- Pulse/bounce animations, custom SVG icon.
- Device targeting and schedule (via data attributes).
- Tooltip option (planned).

## How to use
1. Add **Phone Call Button**, set phone number (digits/plus) and button text.
2. Choose icon and alignment/position.
3. Style colors, radius, shadow, padding, typography.
4. (Pro) Enable floating, device targeting, animation, custom icon.
