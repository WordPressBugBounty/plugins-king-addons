# Image Marquee

Image Marquee creates continuously scrolling rows of images, perfect for logos, partners, or portfolio highlights.

## Free features
- Single-row marquee with horizontal directions (LTR / RTL).
- Per-item image, alt/title fallback, optional hover label, and link.
- Speed, gap, duplicate items for smooth loop, pause on hover/touch.
- Styling for container, items, radius, border, shadow, image sizing/fit, overlay color, and hover scale.

## Pro features
- Vertical directions and multi-row marquee (alternating directions).
- Per-row speed and reverse-on-hover options.
- Extra hover overlays, masks/skins, and advanced filter effects.

## How to use
1. Add the **Image Marquee** widget.
2. Populate items with images, optional labels, and links.
3. Choose direction (LTR/RTL), adjust speed, gap, and duplication for smooth looping.
4. Style item padding, radius, shadow, and image fit/opacity; set overlay and hover label styles.
5. (Pro) Enable vertical or multi-row layouts and advanced hover/mask effects.
