# Sticky Contact Bar

Fixed edge bar with quick-contact shortcuts for chat, call, or social messaging.

## Free features
- WhatsApp, Telegram, Messenger, Phone, Email, or Custom link buttons.
- Default online/offline status labels with optional status badge.
- Edge selection (left/right), vertical alignment (top/middle/bottom), offset and gap.
- Styling for box, status text/dot, and channel buttons (colors, radius, border, shadow).

## Pro features
- Schedule-based online/offline state with timezone offset.
- UTM-based state override for campaign-specific experiences.
- Per-channel visibility rules (online/offline only) and offline status note.
- Device-aware multi-channel setup with status-driven filtering.

## How to use
1. Add **Sticky Contact Bar** to the page.
2. Set the default status text and toggle status visibility.
3. Add channels (label, type, destination, optional prefilled message) and choose visibility.
4. Adjust edge, vertical alignment, and offsets, then style the bar and buttons.
5. (Pro) Enable schedule, configure UTM state, and refine per-channel visibility/device targeting as needed.
