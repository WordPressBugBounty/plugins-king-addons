# Simple Card Grid

A manual grid for images, titles, descriptions, and buttons with safe defaults for spacing and alignment.

## Free features
- Responsive columns and gap controls.
- Card background, border, radius, padding, and shadow toggles.
- Typography and color controls for titles and descriptions.
- Button alignment, text alignment inside the button, border, radius, and shadow toggles.
- Optional whole-card link per item (disabled inside Elementor editor for safety).

## Pro features
- Hover animation presets for cards.
- Extended styling polish for cards and buttons.

## How to use
1. Add the **Simple Card Grid** widget.
2. Create cards with image, title, description, button text/link, and optionally enable whole-card link.
3. Set columns and gap per device in **Layout**.
4. Style card, text, and button; disable shadows if desired.
5. (Pro) Choose a hover animation preset in **Animations**.
