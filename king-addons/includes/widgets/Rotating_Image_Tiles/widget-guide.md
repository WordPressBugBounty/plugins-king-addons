# Rotating Image Tiles

Animated grid of circular image tiles that rotate between multiple images with smooth transitions.

## Free features
- Up to 2 images cycling across tiles.
- Tiles count up to 8 with 2â€“3 columns.
- Autoplay rotation with fixed timing and easing.
- Simple lightbox on click.
- Basic colors, radius, and shadow styling.

## Pro features
- Up to 4 images with titles, descriptions, and links.
- Rotation modes (sequential, reverse, alternate, random) and triggers (autoplay, on hover, on click) with custom interval/easing and pause on hover.
- Per-tile circle masks, delay offsets, and hover scale.
- Click actions: none, lightbox, or custom link; optional mobile animation disable.
- Overlay blend modes, global radius override, hover effects, caption descriptions, and extended style controls.

## How to use
1. Add the **Rotating Image Tiles** widget.
2. (Free) Choose up to 2 images; set tiles count and columns.
3. (Pro) Add up to 4 images with captions/links; configure rotation mode, interval, and behavior.
4. Adjust holder and tile styling; enable captions if needed.
5. (Pro) Fine-tune per-tile circle masks, delays, hover effects, overlay, and mobile behavior.
